블루스택을 이용한 매크로 준비
1. 블루스택 설치 2. 블루스택 우상단 설정-기본설정-해상도1280*720선택
3. 리니지RK 설치
일반 소환석 자동까기 50회
1. RK 소환석 뽑기 화면으로 이동
2. 뽑기.ahk를 AutoHotkey.exe에 drag&drop
3. 50회 횟수 수정이 필요 할 경우 메모장에서 뽑기.ahk 수정 - 주석 editable
스테이지 반복 돌기
1. 스테이지 1-10으로 이동, 우측하단 메뉴에 공격시작이 나올 때 까지 진행
2. gogo.ahk를 AutoHotkey.exe에 drag&drop
3. 스테이지를 수정하고 싶은 경우 gogo.ahk 수정 - 주석 editable
주의사항
1. 사용중에 마우스를 사용 할 경우 매크로가 오작동 합니다.
2. 매크로를 끄고 싶으면 시스템 트레이에서 끄면 되고 마우스가 원치 않는 곳으로 가기 때문에 빠르게 꺼야 합니다. - 끄기 기능 구현 안했네요.
